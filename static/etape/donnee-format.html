Définition Codage d'une donnée

Le codage d'une donnée, est la façon dont cette donnée a été numérisée, c'est à dire représentée sous la forme d'un nombre (donc une suite de bits).
Il est important de connaitre le codage employé quand on veut manipuler une donnée car un même code représente de nombreuses données différentes.
  • 0b01000001 code l'entier 65
  • 0b01000001 code la lettre "A"
  • 0b01000001 code une couleur avec environ 25% d'intensité.
  • ...

Exercice 1 Codage d'une lettre

Si 65 code la lettre "A", quel entier est le code de la lettre "a" ?
Pistes de recherche :
  1. Sur internet avec le mot clé : ASCII
  2. Avec un script python en sachant que ord("A") donne 65.
La réponse est 97.

Définition Format d'un fichier

Le format d'un fichier est la façon d'y organiser des données (avec un codage spécifique).

Beaucoup de formats sont identifiables par l'extention dédiée, associée au nom du fichier :

  • Un fichier de textes peut être au format "TXT", "RTF", "ODT", ...
  • Un fichier image peut être au format "PNG", "JPEG", "GIF", ...
  • Un fichier vidéo peut être au format "MPEG", "MKV", "OGG", ...

Exercice 2 Format d'un fichier texte

Créer un nouveau fichier bonjour.txt et, à l'aide d'un éditeur de texte simple (type Bloc-note), y enregistrer le texte : 
Bonjour !
Ouvrir ce fichier avec un traitement de texte (type LibreOffice) et, sans le modifier, l'enregistrer sous l'extention .rtf.
  1. Quelle est la taille de chaque fichier ?
  2. Pourquoi cette différence ?
  1. Le fichier bonjour.txt fait exactement 9 octets.
    Le fichier bonjour.rtf (enregistré avec LibreOffice) pèse 2 277 octets.
  2. Le format "TXT" n'ajoute aucune information : il y a 9 caractères (espace compris) donc 9 octets.
    Le format "RTF" ajoute des informations : la police de caractère, la couleur, les marges...
Toute entreprise ou particulier qui monte un projet informatique pourrait inventer son propre format de fichier pour répondre à ses besoins. On risquerait vite de ne plus s'y retrouver ! Pour l'éviter, on préfère le plus souvent utiliser des formats génériques qui structurent ces données en tables.
