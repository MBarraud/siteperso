Théorème 1 Les primitives

Soit $f$ une fonction définie sur un intervalle $I$ et de primiteve $F$ sur $I$ :

Toutes les primitives $P$ de $f$ sur $I$ peuvent s'écrire : $P(x) = F(x) + C$ où $C$ est une constante.

Comme $(P-F)'=P'-F'=f-f=0$, $P-F$ est une primitive de $0$, donc constante.
D'où $P-F = C$ puis $P= F + C$, où $C$ est une constante.

Exemple

Si $f(x)=2x$ sur $\mathbb{R}$, ses primitives sont de la forme $F(x) = x^2 + C$, avec $C \in\mathbb{R}$
Dans la pratique, si on demande « les » primitives d'une fonction, il ne faut pas oublier le « $ +C $ » dans la formule.

Théorème 2 Primitives des monômes et inverses

Si $n\in\mathbb{Z}$ avec $n\neq-1$ et $f(x)=x^n$, alors $F(x)=\dfrac{x^{n+1}}{n+1}$ définit une primitive de $f$ sur n'importe quel intervalle de son ensemble de définition.

Ce qui se réécrit pour $n<-1$, en posant $N = -n>1$ : $f(x)=\dfrac{1}{x^N}$, alors $F(x)=\dfrac{-1}{(N-1)x^{N-1}}$

Exemples

  • $f(x) = 1$ donne $F(x)=x$
  • $f(x) = x$ donne $F(x)=\dfrac{x^2}{2}$
  • $f(x) = x^2$ donne $F(x)=\dfrac{x^3}{3}$
  • $f(x) = \dfrac{1}{x}$ ne donne rien avec ce théorème...
  • $f(x)=\dfrac{1}{x^2}$ donne $F(x) = \dfrac{-1}{x}$
  • $f(x)=\dfrac{1}{x^3}$ donne $F(x) = \dfrac{-1}{2x^2}$
( Pour la dernière par exemple : $\dfrac{1}{x^3}=x^{-3}$ donne $\dfrac{x^{-3+1}}{-3+1}=\dfrac{x^{-2}}{-2}=\dfrac{-1}{2x^{2}}$ )

Exercice 1 Primitives des monômes et inverses

Vérifier les exemples du théorème précédant en dérivant, à chaque fois, la fonction $F$.

Dériver une primitive pour la vérifier est un réflexe qu'il faut prendre pour vérifier ses résultats.

  • Pour $F(x)=x$, $F'(x)=1=f(x)$. (Fonction de référence.)
  • Pour $F(x)=\dfrac{x^2}{2} = \dfrac{1}{2}x^2$, comme $(\lambda \times u)' = \lambda u'$ alors $F'(x) = \dfrac{1}{2}2x = x = f(x)$.
  • De même pour $F(x)=\dfrac{x^3}{3}$ : $F'(x) = \dfrac{1}{3}3x^2 = x^2 = f(x)$.
  • Pour $F(x) = \dfrac{-1}{x} = (- 1) \times \dfrac{1}{x}$, on a $F'(x) = (- 1) \times \dfrac{-1}{x^2}=\dfrac{1}{x^2}=f(x)$.
  • Pour $F(x) = -\dfrac{1}{2x^2}$ on peut procéder de même ou utiliser la formule $\left(\dfrac{1}{u}\right)'=\dfrac{-u'}{u^2}$, ce qui donne $F'(x) = -\dfrac{-2\times 2x}{(2x^2)^2}= \dfrac{4x}{4x^4}= \dfrac{1}{x^3}=f(x)$.
