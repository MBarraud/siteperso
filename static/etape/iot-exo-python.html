Définition Entrée/sortie

On appelle instruction d'entrée une instruction qui, à l'exécution, donne la possibilité à l'utilisateur d'entrée une valeur (au clavier, à la souris, avec son doigt sur un écran tactile...)

On appelle instruction de sortie une instruction qui, à l'exécution, produit un signal reconnaissable par l'utilisateur (sur un écran, par des haut-parleurs,...)

Exercice 3.1

Tester les petits programmes suivants :

  1. Pour afficher du texte, il faut l'écrire entre ' ou " : 
    print('Bonjour')
print("Au revoir")
  2. On peut en afficher plusieurs. Chacun sera séparé du suivant par un espace : 
    print('Bonjour', 'Au revoir')
  3. On peut afficher des nombres ou la valeur d'une variable : 
    somme = 12 + 21
print(12,'+',21,'=',somme)
  4. Pour modifier le séparateur, on peut redéfinir le paramètre sep : 
    somme = 12 + 21
print(12,'+',21,'=',somme, sep='')
  5. La fonction ajoute automatiquement un passage à la ligne. 
    print('Bonjour')
    print()
    print('Au revoir')
  6. On peut supprimer le passage à la ligne en modifiant le paramètre end : 
    print('Bonjour', end = '.')
    print('Au revoir', end='.../n')\

Exercice 3.2 Le Chifoumi

On propose le programme suivant :

from random import randint
PIERRE, FEUILLE, CISEAU = 1, 2, 3
JOUEUR, ORDINATEUR = 1, 2

def gagne(joueur,ordinateur):
    resultat = 0
    if joueur == PIERRE:
        if ordinateur == FEUILLE:
            resultat = ORDINATEUR
        elif ordinateur == CISEAU:
            resultat = JOUEUR
    return resultat

score_o, score_j = 0, 0

for i in range(10):
    joueur = int(input("Choisissez 1:Pierre 2:Feuille 3:Ciseau : "))
    ordinateur = randint(1,3)
    gg = gagne(joueur,ordinateur)
    if gg == JOUEUR:
        print("Gagné !")
        score_j = score_j + 1
    if gg == ORDINATEUR:
        score_o = score_o + 1
        print("Perdu !")

print("Score final :")
print("Joueur :",score_j, "Ordinateur :",score_o)
  1. Compléter la fonction gagne pour qu'elle traite bien de toutes les situations de jeu possibles.
  2. Modifier la boucle principale de jeu pour qu'il s'arrête dès qu'un joueur atteint le score de 3.
  3. Ajouter un affichage de ce que joue l'ordinateur.
