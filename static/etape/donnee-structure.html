Définition Open data

Un contenu est dit en open data s’il est librement utilisable, modifiable et partageable par n’importe qui et pour n’importe quelle raison.

Exemples

  • En France, Etalab s’occupe de data.gouv.fr, site de partage des données publiques.
  • Des données sont également disponibles localement, par exemple : data.larochesuryon.fr.
  • Certaines données sont collaboratives, par exemple : fr.openfoodfacts.org autour des produits alimentaires ou openstreetmap.fr sur la cartographie.

Définition Table

Une table est un tableau particulier de données.
  • Le contenu de chaque case du tableau est une valeur.
  • Chaque ligne du tableau est un enregistrement.
  • Les valeurs d’une même colonne du tableau décrivent le même aspect d’un enregistrement.
    Cet aspect est nommé en titre de colonne et appelé descripteur.

Activité 3 Lecture d'un CSV avec Python

Cette activité est facultative mais fortement recommandée pour ceux ayant choisis la spécialité NSI pour l'année prochaine.

On part du fichier CSV chargé à l'activité 2

  1. Renommer votre fichier : retaurants.csv
  2. Recopier le script suivant dans un fichier python, par exemple : retaurants.py 
    import csv

# Ouverture du fichier, en lecture, en mode texte, encodée en UTF-8
with open('restaurants.csv', encoding='utf8') as fichier:
    # Préparation de la lecture du fichier CSV ligne par ligne
    # chaque enregistrements sera utilisable comme une liste de ses valeurs.
    restaurant = csv.reader(fichier, delimiter=';')
    # traitement de chaque enregistrement.
    for restau in restaurant:
        print(restau)
