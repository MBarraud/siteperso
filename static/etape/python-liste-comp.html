Définition Liste en compréhension

L'idée est de simplifier la construction d'une liste en rapprochant son expression du langage naturel.

La syntaxe est la suivante : 

[fonction(item) for item in liste if condition(item)]

Exemples

  • « La liste des images $f(x)$ pour $x$ entier allant de $0$ à $9$ » : 
    [f(x) for x in range(10)]
  • « La liste des multiples de 3 jusqu'à 30 » : 
    [3*n for n in range(11)]
  • « La liste des lettres majuscules (codes ASCII 65 à 90) » : 
    [chr(n) for n in range(65,91)]
  • « La liste des voyelles et celle des consonnes » : 
    voyelle = ['A', 'E', 'I', 'O', 'U', 'Y'] # On a pas trop le choix...
            consonne = [chr(n) for n in range(65,91) if chr(n) not in voyelle]
  • « La liste des multiples de 3 dans une autre liste donnée » : 
    autre = [1, 7, 9, 2, 3, 12]
            [valeur for valeur in autre if valeur%3==0]

Exercice Termes d'une suite récurente simple.

On donne la suite $(u_n)$ définie par $u_0=0$ et, pour tout entier naturel $n$, $u_{n+1}=2u_n+1$.
On veut afficher la liste des $100$ premiers termes à l'aide de Python.

  1. Avec une liste définie en compréhension.
    1. Écrire une fonction Python qui renvoie le terme de rang $N$ de la suite $(u_n)$.
    2. Écrire une liste définie en compréhension qui contient les 100 premiers termes de cette suite.
  2. Avec une liste en extension.
    1. Recopier et compléter le script suivant. 
      def liste_termes(N):
    L=[0]
    for i in range(1,N):
        L.append(...)
    return L
    2. Quelle instruction écrire dans la console pour obtenir la liste des 100 premiers termes de la suite $(u_n)$ ?
  1. Avec une liste définie en compréhension.
    1. Une fonction Python qui renvoie le terme de rang $N$ de la suite $(u_n)$ : 
      def terme(N):
    u=0
    for i in range(1,N):
        u = 2*u+1
    return u
    2. Une liste définie en compréhension qui contient les 100 premiers termes de cette suite : 
      termes = [terme(n) for n in range(100)]
      (Ce n'est pas super malin de faire comme ça ici. On répète trop de fois les mêmes calculs)
  2. Avec une liste en extension.
    1. Je pense que le sujet amène à écrive: 
      def liste_termes(N):
    L=[0]
    for i in range(1,N):
        L.append(terme[i])
    return L
      ce qui n'est pas plus malin que le précédent.
      Je pense que la "bonne" solution serait : 
      def liste_termes(N):
    L=[0]
    for i in range(N-1):
        L.append(2*L[i]+1)
    return L
    2. Dans la console pour obtenir la liste des 100 premiers termes : 
      liste_termes(100)
