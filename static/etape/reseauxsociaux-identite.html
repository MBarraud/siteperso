Définition Réseau social

Les réseaux sociaux sont des applications basées sur les technologies du Web qui offrent un service de mise en relation d’internautes pour ainsi développer des communautés d’intérêts.

Définition Modèle économique

Le modèle économique, c'est la façon qu'ont les applications de gagner de l'argent, sachant qu'elles sempblent le plus souvent en grande partie gratuites pour l'utilisateur.
On peut citer :
  • Le modèle de freemium (seule une partie est gratuite et les options payantes).
  • Les publicités ciblée (par l'utilisation des données de l'utilisateur).
  • La revente de données personnelles.
  • ...

Exercice Fiche d'identité

Choisir un réseaux social dans la liste illustrée de l'exercice précédent en dehors de l'hexagone central et rechercher :
  • Son logo.
  • L'année de sa création (et éventuellement les dates de ses mises à jour importantes).
  • Le nom du/des créateur(s).
  • Son pays d'origine.
  • S'il a été racheté par un groupe plus important.
  • Son nombre actuel d'utilisateurs/membres/abonnés.
  • Son/ses usage(s).
  • Son modèle économique.
  • Son chiffre d'affaire.
  • Le nombre de ses employés.
Rassembler ces informations sur une page HTML mise en forme par un fichier CSS et poster votre dossier de travail (sous la forme d'une archive sur pronote).
