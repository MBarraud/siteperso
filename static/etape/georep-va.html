Définition Distance sur un axe

On note $\vert b - a \vert$ la distance entre le point d’abscisse $a$ et le point d’abscisse $b$.
On dit aussi par abus de langage que $\vert b - a \vert$ est la distance entre a et b.
Le résultat est celui de la différence $b - a$ ou $a - b$ qui est positive.

Exercice 3.14 (voir aussi 67 p.29)

Dans chaque cas, écrire avec la notation $\vert \ldots \vert $ et calculer la distance entre les nombres réels :

  1. $-2$ et $7$
  2. $\sqrt{2}$ et $-\sqrt{2}$
  3. $\dfrac{2}{3}$ et $\frac{3}{4}$
  1. $\vert -2 - 7\vert = \vert 7 - (-2)\vert$ = 7+2=9
  2. $\vert\sqrt{2} - (-\sqrt{2})\vert =2\sqrt{2}$
  3. $\vert\frac{2}{3}-\frac{3}{4}\vert=\vert\frac{3}{4}-\frac{2}{3}\vert=\frac{9}{12}-\frac{8}{12}=\frac{1}{12}$

Exercice 3.15 (voir aussi 68 p.29)

$x$ désigne un nombre réel. Dans chaque cas, interpréter en terme de distance en deux nombres réels :

  1. $\vert x - 2\vert$
  2. $\vert x + 2\vert$
  3. $\vert 3 - x\vert$
  4. $\vert 3 + x\vert$
  5. $\vert -4 - x\vert$
  1. $\vert x - 2\vert$ est la distance entre $2$ et $x$.
  2. $\vert x + 2\vert = \vert x -(-2)\vert$ est la distance entre $-2$ et $x$.
  3. $\vert 3 - x\vert$ est la distance entre $x$ et $3$.
  4. $\vert 3 + x\vert=\vert x -(-3)\vert$ est la distance entre $-3$ et $x$.
  5. $\vert -4 - x\vert$ est la distance entre $x$ et $-4$.

Exercice 3.16 (voir aussi 69 p.29)

Sur une droite graduée, A, B, C et D sont les points d'abscisses respectivement $-3$, $-\frac{1}{3}$, $0$ et $2$.

Calculer les distances AB, AC, AD, BC, BD et CD.

Il est toujours recommandé de faire une figure !

Les caculs se font par valeur absolue de la différence, c'est à dire la diffrence du plus grand par le plus petit nombre :

  • $AB = \vert -\frac{1}{3} - (-3)\vert = -\frac{1}{3} +3 = \frac{8}{3}$
  • $AC = \vert 0 - (-3)\vert = 0 +3 = 3$
  • $AD = \vert 2 - (-3)\vert = 2 +3 = 5$
  • $BC = \vert 0 - (-\frac{1}{3})\vert = \frac{1}{3}$
  • $BD = \vert 2 - (-\frac{1}{3})\vert = 2 + \frac{1}{3} = \frac{7}{3}$
  • $CD = \vert 2 - 0\vert = 2 $

Exercice 3.17 (voir aussi 70 p.29)

Dans chaque cas, écrire le nombre sans la notation $\vert \ldots \vert $ :

  1. $\vert \pi - 3\vert$
  2. $\vert 2 - \sqrt{2} \vert$
  3. $\vert -2 - \pi \vert$
  4. $\vert \sqrt{3} + 1\vert$

Les caculs de valeur absolue d'une différence se font par la diffrence du plus grand par le plus petit nombre :

  1. $\vert \pi - 3\vert=\pi - 3$ car $\pi > 3$
  2. $\vert 2 - \sqrt{2} \vert = 2 - \sqrt{2}$ car $2 > \sqrt{2}$
  3. $\vert -2 - \pi \vert = \vert \pi - (-2) \vert = \pi +2$
  4. $\vert \sqrt{3} + 1\vert = \vert \sqrt{3} - (- 1)\vert = \sqrt{3} - (- 1) = \sqrt{3} +1$

Exercice 3.18

Résoudre l'équation $\vert x - 3\vert = 4$

Première solution (géométrique : faire une figure) : $\vert x - 3\vert = 4$ signifie que la distance entre $3$ et $x$ est de 4. Donc $x = 3 + 4 = 7 $ ou, de l'autre côté : $x = 3 - 4 = -1 $

Seconde solution (par la définition) : $\vert x - 3\vert$ vaut soit $x - 3$, soit $3-x$ selon si $x$ est plus grand ou plus petit que 3.
Dans le premier cas, cela donne $x - 3 = 4$ donc $x = 7$ et dans le second $3 - x = 4$ donc $-1 = x$

Troisième solution (algébrique) : $\vert x - 3\vert$ peut se réécrire $\sqrt{( x - 3)^2}$. L'équation se réécrit donc $\sqrt{( x - 3)^2}=4$. Or comparer deux nombres positifs revient à comparer leurs carrés, c'est à dire $( x - 3)^2=16$. C'est une équation carré dont on sait qu'elle admet deux solutions : $x - 3=4$ ou $x - 3=-4$. On retrouve donc encore $x=7$ ou $x=-1$.

Bilan :
La dernière méthode pouvant se généraliser, on peut retenir et utiliser le résultat suivant :

$\vert A(x)\vert=a\Leftrightarrow A(x) =a \text{ ou } A(x)=-a$

Propriété (Exercice 3.19)

La distance de $x$ à $y$ est $\vert y - x \vert = \left\lbrace\begin{matrix}y-x & si & x\leqslant y\\ x-y & si & y\leqslant x\end{matrix}\right.$

La distance de $x$ à 0 est $\vert x \vert = \left\lbrace\begin{matrix}x & si & 0 \leqslant x\\-x & si & x\leqslant0\end{matrix}\right.$

On lit donc $\vert x \vert$ : valeur absolue de $x$
